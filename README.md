# Retro Flappy Bird - Mobile & Desktop

A high-fidelity Flappy Bird clone built with pure HTML5, CSS3, and JavaScript. This project focuses on smooth physics, responsive design for mobile devices, and a retro pixel-art aesthetic.

## 🚀 Features
* **Cross-Platform:** Perfectly playable on mobile (touch) and desktop (mouse).
* **High Graphics:** Features procedural parallax clouds, dynamic bird rotation based on velocity, and thick-stroke "meme" style pipes.
* **Optimized Performance:** Uses `requestAnimationFrame` for smooth 60FPS gameplay.
* **Zero Dependencies:** Built entirely with vanilla JavaScript and the Canvas API.

## 🛠️ Technical Highlights
* **Physics Engine:** Custom gravity and velocity-based movement.
* **Collision Detection:** Precise AABB (Axis-Aligned Bounding Box) logic.
* **Mobile First:** Includes viewport meta-tags and touch event `preventDefault` to eliminate mobile tap latency.

## 🕹️ How to Play
1. Clone this repository.
2. Open `main.html` in any web browser.
3. **Mobile:** Tap the screen to flap.
4. **Desktop:** Click the mouse to flap.
